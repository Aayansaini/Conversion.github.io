What is Binary Number?

A number system where a number is represented by using only two digits (0 and 1) with a base 2 is called a binary number system. For example, 10012 is a binary number.

What is Decimal Number?

A decimal is a number that consists of a whole and a fractional part. Decimal numbers lie between integers and represent numerical value for quantities that are whole plus some part of a whole. Fractional form: In fraction form, we can write that there is one and one-half of a pizza. That is 1 1 2 pizza.
